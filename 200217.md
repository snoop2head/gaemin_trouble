# Django Migration History Fuck up 됐을 때

- ./app/migration 디렉토리는 절대 삭제하면 안 된다. 

  ```shell 
  python manage.py <command>
  ```

  위 커맨드로 할 수 있는 게 하나도 없다. 

- 차라리 .db.sqlite3는 삭제하고, 

  ```shell 
  python manage.py makemigrations
  python manage.py migrate
  ```

  이렇게 진행하는 편이 나은 것 같다.
